# Stoic Operative Style Guide (Prometheus)

## Non-negotiables
- High contrast, cinematic darkness, controlled red accents
- Silhouette / statue / operative shadow — never weak posture
- Bold short text (≤4 words on thumbnails)
- Grain + tactical grid allowed; pastel softness forbidden
- Operative Score ≥ 85 to publish; thumbnail ≥ 90

## Enhancement order on rework
1. Increase contrast / sharpen
2. Darken midtones, lift subject edge
3. Amplify blood-red keyword
4. Reduce clutter / secondary text
5. Regenerate if score still < threshold
