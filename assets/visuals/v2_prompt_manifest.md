# V2 Visual Prompt Manifest

All V2 images were generated as original assets with OpenAI image generation for this production. The shared art direction was cinematic realism, 16:9 composition, restrained black/charcoal palette, controlled red accents, natural anatomy, no logos, no watermarks, no weapons, and no embedded text unless explicitly specified.

## Thumbnail

Final concept: a fractured classical marble head on a black background, hard side lighting, red light glowing through the fracture, generous negative space, and only the exact three-line typographic treatment `DANGER`, red `=`, `OPTIONS`. The image communicates calm capability rather than violence.
